# 이미지 파일 안내

## 필요한 이미지 파일

### 1. og-image.png (필수)
- **위치**: `public/og-image.png`
- **크기**: 1200x630px (권장)
- **용도**: 카카오톡, 페이스북 등 SNS 링크 공유 시 미리보기 이미지
- **내용**: 평안상조 로고와 함께 "평안상조 - 품격있는 장례 서비스" 텍스트 포함

### 2. logo.svg (선택)
- **위치**: `public/logo.svg`
- **크기**: 512x512px (권장)
- **용도**: 파비콘, 기본 로고
- **현재**: SVG 버전이 생성되어 있습니다

## 이미지 생성 방법

### 방법 1: 온라인 도구 사용
1. Canva, Figma 등 디자인 도구 사용
2. 1200x630px 캔버스 생성
3. 평안상조 로고 추가
4. "평안상조 - 품격있는 장례 서비스" 텍스트 추가
5. PNG로 내보내기 → `public/og-image.png`에 저장

### 방법 2: 기존 로고 이미지 사용
1. 제공하신 평안상조 로고 이미지 준비
2. 이미지 편집 도구로 1200x630px 크기로 조정
3. 배경 추가 (선택사항)
4. `public/og-image.png`로 저장

## 이미지 최적화 팁

- 파일 크기: 1MB 이하 권장
- 형식: PNG (투명 배경) 또는 JPG
- 해상도: 1200x630px (Open Graph 표준)
- 색상: sRGB 색공간 사용

## 테스트 방법

1. 이미지 파일을 `public/` 폴더에 추가
2. 개발 서버 재시작: `npm run dev`
3. 브라우저에서 `http://localhost:3000/og-image.png` 접속하여 확인
4. [Open Graph 디버거](https://developers.facebook.com/tools/debug/)에서 테스트
5. 카카오톡에서 링크 공유하여 미리보기 확인
